STUDY JAM TAREA 3: CALCULADORA FUNCIONAL
========================

##DESCRIPCIÓN
Esta es una aplicación Android de una calculadora funcional que viene preinstalada en la ROM
original de la version 6.0 de Android (Marshallow)

Requerimientos
------------

  * Android Studio 2.2.2
  * Gradle 2.2.2
  * minSdkVersion 14
